# Example Prompts for Inspiration

## How to use

You don't need to copy these prompts — Claude writes prompts automatically based on the website you provide.
This is just to show you the format and style of prompts that work well.

## Tips

- Replace generic product descriptions with the EXACT packaging description from the client's website
- Adapt text overlay to the target market's language
- Vary the audience — don't make 50 identical scenes with elderly people
- Always include price/discount and CTA in text overlay
- Add social proof where possible (number of customers, ratings, reviews)
